UI sound placeholders
=====================

Drop optional UI sound files here. If a file is missing, the game uses a
built-in WebAudio beep instead, so these files are not required.

Expected filenames:

  button-click.mp3   -> every button press
  countdown-3.mp3    -> Start countdown "3"
  countdown-2.mp3    -> Start countdown "2"
  countdown-1.mp3    -> Start countdown "1"
  countdown-go.mp3   -> Start countdown "GO"

Keep clips short:

- button-click.mp3: about 0.05-0.15 seconds
- countdown clips: about 0.2-0.6 seconds
