威尼斯 3D 划船體感遊戲測試工具

這版遊戲改成 Three.js 3D 威尼斯 gondola 闖關。孩子站在 iPad 鏡頭前，用大幅度左右划船動作控制畫面上的 3D 小船， 穿過運河上的發光任務門。彩繪玻璃作品可以當作闖關前展示的船票或隊伍徽章。

開啟測試模式 開啟正式遊戲

孩子怎麼玩

  1. 老師用 iPad 開正式遊戲，投影到大螢幕。
  2. 孩子站在鏡頭前，雙手像拿船槳一樣左右划。
  3. 左邊動作和右邊動作會影響小船方向與速度。
  4. 目標是穿過 Ciao Gate、Rialto Bridge、Glass Window、Pizza Stop、Museum Dock。
  5. 每過一關，全班一起念畫面下方英文句子。

測試模式怎麼用

  1. 按「開啟測試模式」。
  2. 按 Start。
  3. 不用開相機，按 Left / Right / Row / Boost / Brake 測試小船控制。
  4. 如果現場光線不好，老師可以用 Pass 讓小組直接過關。

控制意思

Left

模擬左側划船動作，讓小船往一邊轉。

Right

模擬右側划船動作，讓小船往另一邊轉。

Row

左右一起划，讓小船穩定往前。

Boost

快速加速，用來測試闖關節奏。

Brake

減速，測試孩子划太快時畫面是否還好控制。

Pass

老師手動過關，給現場備援。
